#  Bank Analytics & Dashboard – Data Analytics Project

##  Overview
This project focuses on analyzing banking and loan data to monitor portfolio performance, assess risk, and enhance financial decision-making. It covers end-to-end data analytics including data extraction, transformation, KPI tracking, and dashboard development.

---

##  Objective
To analyze loan and banking data in order to:
- Monitor portfolio performance  
- Assess financial risk  
- Improve lending and business decisions  

---

##  Tools & Technologies
- SQL  
- Excel  
- Power BI  
- Tableau  

---

##  Business Need
The organization required detailed insights into:
- Loan performance  
- Customer segmentation  
- Risk metrics (defaults and delinquencies)  

To optimize lending strategies and minimize financial risk.

---

##  Data Processing & ETL
- Extracted and processed large datasets using SQL and Excel  
- Performed data cleaning, transformation, and data modeling  
- Structured datasets for efficient analysis and reporting  

---

##  Data Preparation
- Segmented data by:
  - Customer demographics  
  - Product categories  
  - Loan status  
- Prepared datasets for portfolio and risk analysis  

---

##  Analysis & KPIs
- Defined and tracked key business metrics:
  - Loan disbursement  
  - Collections  
  - Interest revenue  
- Conducted descriptive and diagnostic analysis of loan portfolios  
- Performed risk analysis using:
  - Default rates  
  - Delinquency rates  
  - Credit grade segmentation  

---

##  Visualization & Reporting
- Built interactive dashboards using:
  - Power BI  
  - Tableau  
- Visualized:
  - Loan disbursement trends  
  - Repayment patterns  
  - Risk distribution  
- Created reports for stakeholders to support decision-making  

---

##  Key Outcomes
- Delivered insights on loan portfolio performance  
- Identified high-risk segments and trends  
- Improved decision-making through data-driven risk analysis
